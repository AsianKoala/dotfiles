## Installation
Clone and run `bash install.sh` 
If `~/.local/bin` is not in your path already, add it with:
`export PATH="~/.local/bin:\$PATH"`

Now run `vis`!

## Dependencies
- [vis](https://github.com/dpayne/cli-visualizer)
- [pywal](https://github.com/dylanaraps/pywal)
- awk
- bc
- sed
